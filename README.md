## Featured project

### [QGIS InSAR Time Series Viewer](https://github.com/vmendes93/qgis-insar-timeseries-viewer)

A QGIS plugin for exploring, averaging, validating and exporting point-based InSAR displacement time series.

This project reflects the kind of work I care about most: bringing together **geological reasoning**, **geotechnical monitoring**, **remote sensing data** and **Python-based tooling** in a practical interface for spatial interpretation.

**Focus:** InSAR time series · QGIS plugin development · PyQGIS · displacement monitoring · data validation · technical visualization
